# OpenLabor Community

Open source skills, docs, and resources for the [Labor](https://labor.so) AI workforce platform.

## What's here

```
skills/          — AI-powered skills anyone can use, contribute to, or fork
docs/            — guides, references, and how-tos
templates/       — starter configs and boilerplates
examples/        — usage examples and demos
```

## Skills

Skills are structured prompts that give AI agents specialized capabilities. Each skill includes instructions, scoring frameworks, and API integrations.

| Skill | Description |
|-------|-------------|
| [domain-advisor](skills/domain-advisor/) | Find, check, and evaluate domain names with business-grade analysis |

## Contributing

We welcome contributions! To add a skill:

1. Fork this repo
2. Create a folder under `skills/your-skill-name/`
3. Add a `SKILL.md` following the format in existing skills
4. Submit a PR
