# Daily UI Component Audit

Check design system consistency and flag outdated or non-compliant components.

## Prompt

Use #Design system management to scan all active UI components against the current design system spec. Use #Image editing to compare component renders with approved design tokens (colors, spacing, typography). Apply #Brand assets to verify logo and icon usage follows placement guidelines. Use #Social media graphics templates to check that reusable components are up to date. Flag any components that deviate from the system and draft update recommendations with visual diffs.
