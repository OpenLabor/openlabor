# Weekly Dependency Update

Check for outdated packages, review security patches, and draft upgrade pull requests.

## Prompt

Use #CI/CD pipelines integration to run dependency audit across all project packages. Use #Code review to identify outdated dependencies with available minor and major updates. Apply #Security audits to flag any dependencies with known CVEs or security advisories. Use #Bug fixing to test compatibility of critical updates and draft upgrade PRs with changelog summaries. Compile a dependency health report listing outdated packages, security risks, and recommended upgrade priority.
