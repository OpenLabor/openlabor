# Quarterly Naming Workshop

Brainstorm product and feature names, check availability, and test resonance.

## Prompt

Use #Brand naming to brainstorm 20-30 candidate names for upcoming products, features, or initiatives based on brand strategy and positioning. Apply #Trademark screening to filter candidates against existing trademarks in relevant classes. Use #Domain research to check domain availability for shortlisted names across key TLDs (.com, .io, .co, .ai). Use #Brand strategy to evaluate each name for target audience resonance, international viability, and brand architecture fit. Compile a naming workshop output with top 5 recommended names, each with trademark clearance status, domain availability, and resonance scoring.
