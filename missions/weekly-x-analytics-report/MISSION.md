# Weekly X Analytics Report

Compile follower growth, top posts, engagement rates, and actionable recommendations for X.

Use #X analytics & reporting to pull the full week's performance data including impressions, engagement rate, follower growth, and profile visits. Use #Viral content strategy to benchmark against industry averages. Compile a report with top 5 posts, key metrics, and 3 actionable recommendations for next week.
