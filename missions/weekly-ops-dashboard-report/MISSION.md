# Weekly Ops Dashboard Report

Compile a comprehensive weekly operations report with KPIs, blockers, and action items.

## Prompt

Use #KPI tracking to pull this week's performance metrics across all departments. Use #Reporting dashboards to compile data into a visual executive summary. Apply #Team coordination to gather status updates and identify blockers from each team. Use #Workflow optimization to flag any processes that showed inefficiency this week. Use #SOPs to recommend process improvements. Output a structured report with KPI trends, blockers, wins, and action items for the coming week.
