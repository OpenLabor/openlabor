# Daily Team Standup Summary

Collect async standups from all teams, compile blockers, progress, and key updates.

## Prompt

Use #Team coordination to collect async standup updates from all team channels and project management tools. Use #KPI tracking to pull key daily metrics for each department. Apply #Workflow optimization to identify any new blockers or dependencies that could slow progress. Use #Reporting dashboards to compile a concise summary with per-team progress, blockers, and items needing escalation. Output a structured daily briefing that can be shared with leadership in under 2 minutes.
