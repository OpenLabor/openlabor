# Weekly Revenue Dashboard

Compile revenue metrics, cohort analysis, and churn trends into an executive dashboard.

## Prompt

Use #SQL queries to pull weekly revenue data including MRR, ARR, new revenue, expansion, and churn by cohort. Apply #Trend analysis to compare this week's revenue metrics against trailing 4-week and 12-week averages. Use #Forecasting to project end-of-month and end-of-quarter revenue based on current trajectory. Use #Anomaly detection to flag any unusual churn spikes or conversion rate drops. Compile an executive revenue dashboard with visualizations, cohort retention curves, and 3 key takeaways for leadership.
