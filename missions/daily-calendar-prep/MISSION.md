# Daily Calendar Prep

Review tomorrow's meetings, send prep briefs, and confirm attendance.

## Prompt

Use #Email triage to scan tomorrow's calendar for all scheduled meetings, calls, and events. Use #Email drafting to prepare brief agendas or talking points for each meeting based on recent email threads and project updates. Apply #Follow-up tracking to check if any pre-meeting action items are still outstanding. Use #Auto-replies to send meeting confirmations and prep materials to all attendees. Output a daily calendar briefing with meeting times, attendees, agendas, and any items requiring preparation.
