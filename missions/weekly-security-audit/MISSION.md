# Weekly Security Audit

Perform a comprehensive security scan of the codebase and generate a vulnerability report.

## Prompt

Use #Security audits to run a full scan of the codebase for known vulnerability patterns. Check dependencies for CVEs using #CI/CD pipelines integration. Use #Architecture planning to evaluate if any new code introduces architectural security risks. Apply #Bug fixing to draft patches for critical and high-severity issues. Compile a prioritized vulnerability report with severity ratings, affected files, and recommended remediations.
