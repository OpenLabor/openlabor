# Weekly Brand Asset Inventory

Catalog all logo usage across platforms and flag inconsistencies or unauthorized usage.

## Prompt

Use #Brand assets to scan all platforms (website, social profiles, email templates, partner sites) for current logo usage. Use #Logo concepts to compare each instance against the approved logo versions and placement rules. Apply #Style guide creation to verify correct color treatments, clear space, and minimum size compliance. Use #Icon/mark design to check favicon and app icon consistency. Compile an inventory report listing all logo placements, flagging any that are outdated, distorted, or non-compliant.
